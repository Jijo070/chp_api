_find_close_cached_query(parsed_query, potential_objs, threshold=None)¶

This fuction will find a query that matches exactly genes, drugs, disease, outcome name, outcome operator, but will try to find a close query in the outcome value given by the threshold.

Parameters

  • query (dict) – The query.

  • threshold (float) – The fraction (distance from outcome value) / outcome_value. If this fraction is less than the threshold it will not return that query. Default None, in which no threshold of this fraction is acceptable so just return None.
